# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.3.x   | Yes       |
| 0.2.x   | Best effort |
| < 0.2   | No        |

## Reporting a Vulnerability

If you discover a potential vulnerability in fixr, please do not
open a public issue.

Instead, contact the maintainers through a private channel and provide:

- A description of the issue
- Steps to reproduce
- The affected version and commit hash
- Any suggested mitigation

We aim to acknowledge reports within 72 hours and publish a fix or an advisory
within 14 days, depending on severity.

## Scope

This policy covers the Rust programs under ```programs/``` and
```crates/```, and the TypeScript SDK under ```sdk/```.

Out-of-scope:
- Third-party RPC providers
- Wallet implementations
- The website under https://fixr.red
